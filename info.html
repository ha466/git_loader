🛠️ Installation & ⚠️ Limitations

Are you ready for the complex build steps?

  1. Clone or download this repo.
  2. Double-click index.html.
  3. That's it. No npm install, no Webpack nightmares, no dependency hell. Just Vanilla JS, Tailwind via CDN, and pure vibes.
  • Rate Limits: If you are unauthenticated and downloading rapidly, GitHub's API might throw a 403 Rate Limit Exceeded tantrum. Wait a few minutes.
  • Private Repos: Doesn't work for private repositories (client-side app, doesn't ask for tokens... yet).
